### ðŸš€ Installation

Clone the repository.

```bash
git clone https://github.com/slydragonn/dotfiles.git
```

Move nvim directory from `/dotfiles/nvim/packer/` to

- **Windows**: `$env:LOCALAPPDATA\nvim\`

- **Linux**: `~/.config/nvim/`

and run the command:

```bash
nvim +PackerSync
```

or in Neovim Editor:

```bash
:PackerSync
```
